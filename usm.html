Overview

A production tool built to make Unity audio faster and cleaner.

USM reduces repetitive audio code with simple playback calls, visual editor controls, timeline-style composition, and reusable configuration patterns. It is built for developers who want fast setup without losing control.

This project shows editor tooling, API design, documentation mindset, and a product-quality Unity Asset Store workflow.
